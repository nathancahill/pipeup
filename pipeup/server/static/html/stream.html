PipeUp

PipeUp is a command line tool for piping to a webpage:
$ tail -f /var/log/nginx/access.log | pipeup

Start using PipeUp by installing the client with pip:
$ pip install pipeup

Both the PipeUp client and server are MIT licensed and available on Github.
